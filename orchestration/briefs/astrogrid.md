# AstroGrid Brief

## Truth
AstroGrid is its own entity.

Not a GRID feature.
Not a decorative surface.
Not a skin.

## Shared Ground
Allowed:
- same database
- selected GRID prediction outputs
- shared infra

Forbidden:
- shared identity
- shared tone
- shared UX logic
- hidden coupling

- readings

- agent access

- Users can isolate, combine, or intersect lenses.
