# AstroGrid Visualization

## Law
Visuals reveal computed state.

They do not decorate absence.

## Primary Surfaces
- Orrery
- Radial sky
- Aspect field
- Event band
- Object table
- Inspector

## Orrery
Purpose:
- spatial relation
- orbital rhythm
- body motion

Shows:
- body positions
- orbital paths
- motion direction
- distance tiers
- highlighted active objects

## Radial Sky
Purpose:
- zodiac structure
- sign occupancy
- angle reading
- dense comparison

Shows:
- 360 degree wheel
- sign bands
- body glyphs
- degree markers
- nodes
- optional fixed stars

## Aspect Field
Purpose:
- relationship geometry
- tension
- harmony
- convergence

Shows:
- aspect chords
- aspect type
- orb strength
- applying vs separating
- hidden or muted lens overlays later

## Event Band
Purpose:
- timing
- ingress
- stations
- eclipses
- cycle thresholds

Shows:
- near events
- exact times
- active windows
- resonance clusters

## Object Table
Purpose:
- numbers
- truth surface

Shows per object:
- longitude
- latitude
- right ascension
- declination
- distance
- speed
- retrograde state
- sign
- degree
- precision

## Inspector
Purpose:
- one object in full

Shows:
- object metadata
- current position
- motion state
- active aspects
- cycle notes
- linked readings later

## Visual Grammar
- orbit = path
- glyph = body
- chord = aspect
- pulse = event
- ring = cycle
- haze = uncertainty or lower precision

## Precision Display
The UI should expose source trust.

Ways:
- line sharpness
- label treatment
- haze level
- explicit precision mark

Do not hide uncertainty.

## Default Visibility
Default:
- core set only

Optional:
- second ring

Advanced:
- research objects

## Anti-Patterns
- fake stars for mood
- unlabeled geometry
- card soup instead of instrument
- charts detached from source data
- mystical filler where numbers should be
