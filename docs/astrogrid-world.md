# AstroGrid World

## Rule
One scene graph.

Not one globe per idea.

## Engine Choice
- celestial truth: ephemeris + orbital propagation
- world engine: Cesium
- local instrument overlays: React Three Fiber
- Earth vector overlays: MapLibre

## Why
- vector maps are correct for Earth overlays
- they are not the core world model
- Earth, Moon, Mars, and orbital shells need one spatial spine
- satellites and transfer corridors need time-dynamic geometry
- capital flows must bind to places, orbits, corridors, and assets
