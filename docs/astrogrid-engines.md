## Lens Modes

### Solo
One engine.
One voice.
No bleed.

### Chorus
Many engines.
Side by side.
No forced union.

### Intersection
Show overlap.
Show fracture.
Show what survives the cut.

### Shadow
One primary lens.
Others dimmed.
Present, not loud.

## Lens Permission
- User chooses the lens.
- No silent mixing.
- No tradition speaks for another.
- Intersection is explicit, not default.
- Muted engines remain logged.
- Forbidden lenses must not leak into output.

## Engine Output
Each engine should emit:
- reading
- omen
- prediction
- rationale
- confidence
- horizon
- feature trace

## Logging
Every engine speaks into its own ledger.

Log:
- timestamp
- engine id
- lens mode
- input sky state
- derived features used
- output text
- prediction claims
- confidence
- horizon
- later outcome

Keep logs append-only.
Keep them separate.

## Why Logs Matter
No ledger.
No memory.
No calibration.

The Seer needs history to judge signal.
